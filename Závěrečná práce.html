Tato stránka představuje web stavebnin, který obsahuje vše potřebné pro školní zadání i pro dobrý vizuální dojem.

Proč právě tento návrh

Web kombinuje přehledné sekce, profesionální barvy, jasnou navigaci a jednoduché ovládání. Zákazník si prohlédne nabídku a pomocí formuláře odešle poptávku.

Je to vhodné řešení pro firmu, která nechce klasický e-shop, ale chce získávat kontakty a individuální požadavky.
